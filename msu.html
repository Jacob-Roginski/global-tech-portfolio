Michigan State University

UI/UX Designer | Fall 2025

Overview

A comprehensive UI/UX design project for Michigan State University focused on enhancing the digital experience for students, faculty, and staff. This project demonstrates user-centered design principles and systematic approach to solving complex university interface challenges.

The Challenge

The university needed a more intuitive and accessible digital interface to serve a diverse user base. The main challenges were:

  • Creating a unified experience across multiple user types
  • Improving information architecture for easy navigation
  • Ensuring accessibility compliance for all users
  • Maintaining brand consistency while modernizing the design

My Approach

User Research

Conducted extensive research with students, faculty, and administrators to understand diverse needs.

Information Architecture

Restructured content hierarchy to improve findability and reduce cognitive load.

Visual Design

Created a modern, accessible design system aligned with MSU brand guidelines.

Iterative Testing

Validated design decisions through multiple rounds of user testing and feedback.

Key Features

Personalized Dashboard

Customized interface for different user roles with relevant information and quick actions.

Streamlined Navigation

Intuitive menu structure and search functionality for easy access to resources.

Mobile-First Design

Responsive interface optimized for mobile devices used by students on campus.

Accessibility Features

WCAG compliant design with support for screen readers and keyboard navigation.

Outcome & Learnings

This project reinforced the importance of designing for diverse user groups with varying technical proficiency and needs. By conducting thorough user research and iterative testing, we created a solution that improved the digital experience across the university community.

Key learnings included the value of strong information architecture, the importance of accessibility from the start, and how effective collaboration with stakeholders leads to better design outcomes.
